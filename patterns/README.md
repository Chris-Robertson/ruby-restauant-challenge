## ATM

### Skills

- Classes: `class`
- Subclassing: `class B < A`
- Overriding methods: `def make_tile`, `def to_s`
- Measuring array & string length
- Maths: `/`
- Loops: `.times`, `.each`
- Arrays: `<<`, `.join`
- System commands: `system`, backticks
